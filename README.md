# UMBRAL 3 — Sitio web

Landing page de UMBRAL 3. Una sola página, estática (HTML + CSS, sin dependencias).

## Archivos
- `index.html` — la página completa (estilos y scripts incluidos en el mismo archivo).

## Publicación
El sitio se publica con **Netlify**, conectado a este repositorio.
Cada vez que se hace `push` a la rama principal, Netlify actualiza la web automáticamente.

## Cómo editar
1. Editar `index.html`.
2. Hacer `commit` y `push`.
3. Netlify republica solo en unos segundos.

## Pendientes / a completar en `index.html`
- Mail de contacto: `hola@umbral3.com`
- Teléfono en el pie de página.
- Conectar el dominio propio (`umbral3.com`) desde el panel de Netlify.
